- Merge Sort -

A comparison-based sorting with a complexitiy of O(n log n), it sorts the data with a divide and conqure method, where datas are divided into n number of arrays and merge slowly by comparing elements of each array one at a time, as seen in the animation.

This was a project done during my holiday where I wanted to expand my skills and learn something new, upon learning sorting techniques, I thought it would be more fun and interesting to create an 'animatable' version while creating a deeper understanding of the sorting technique. Hope this would help you in some way or another.
