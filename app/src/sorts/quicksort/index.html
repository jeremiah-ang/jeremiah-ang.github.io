- Quick Sort -

Divide-and-conquere
Quicksort is an sorting algorithm with worst-case of O(n^2) but an average case of O(n log n).
It is a comparision sort and it is not a stable sort.

The sort is carried out by selecting a pivot among a partition, in which any value smaller than the pivot would be placed in the left and any value bigger would be placed in the right side, the values in the left and right side would by its on become another partition, as it goes on, it eventually reaches multiple partitions of size 1, at that, the sorting is complete.
