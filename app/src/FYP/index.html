StopCheck is a location-based
reminder mobile application
which provides reminder with
just 3 simple steps!

Search or pin location

Set a location that you want to be notified when entering or leaving

Simply search or tap on the map to drop a pin on the location

When done, the location name and address will be displayed

Set alarm to location

Write notes and add then to the location

Set alarm to ring on selected days and upon enter or leaving a location

Check when alarm rings

Stop to check when alarm rings to make sure you did not forget anything when you enter or leave your selected location!

With StopCheck you'll never forget anything again!
